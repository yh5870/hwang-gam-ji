# API 키 보안 안내

## 현재 구조

- API 키는 `.env`의 `VITE_KMA_API_KEY`에 저장
- Vite 빌드 시 클라이언트 번들에 **노출됨** (브라우저 개발자도구에서 확인 가능)
- 공공데이터포털 API는 일일 호출 제한으로 남용 방지

## 권장 사항

### 개발 / 테스트
- `.env` 파일을 **절대 git에 커밋하지 마세요** (`.gitignore`에 포함됨)
- `.env.example`만 저장소에 포함 (실제 키 없음)

### 운영 배포
- **백엔드 프록시** 사용 권장: 프론트 → 백엔드 → 기상청 API
- API 키는 백엔드 환경변수로만 관리
- 프론트엔드에는 키를 두지 않음
