# 황감지 웹사이트 시각 개선 제안 (5군데)

전반적으로 심심한 느낌을 줄이기 위해 추가/보완하면 좋을 영역을 정리했습니다.

---

## 1. 배경 — 은은한 별·라인 패턴

**현재:** 그라데이션 + blur 원 2개만 있음.

**제안:** 밤하늘 느낌을 위한 은은한 효과 추가
- `noise`/`grain` texture (반투명 오버레이)
- 상단에 아주 작은 점/별 스캐터 (0.5~1px, opacity 낮게)
- 또는 미세한 그리드 라인 (0.5px, 5% opacity)로 “데이터 패널” 느낌

→ 배경에 깊이감과 야경 느낌을 더할 수 있음.

---

## 2. 카드(박스) — 테두리 글로우·그라데이션

**현재:** 단색 border, 단순 shadow.

**제안:** 카드가 살짝 빛나는 느낌
- 상단/측면에 아주 얇은 `box-shadow` 글로우 (예: cyan 0 0 20px rgba(34,211,238,0.08))
- 카드 테두리를 `linear-gradient`로 처리 (상단만 살짝 밝게)
- 호버 시 border/glow 강도 증가

→ 메인 카드, 상세 지표 카드, 설명 박스에 모두 적용 가능.

---

## 3. 설명 박스(Description Box) — 배경·구분감 강화

**현재:** `bg-black/40`, 거의 단색.

**제안:**
- 좌측 또는 하단에 accent color 그라데이션 라인 추가
- `backdrop-blur` 강화로 “유리” 느낌
- 또는 좌측에 얇은 세로 바(2~3px) + 보라/시안 글로우

→ 메인 메시지가 들어가는 박스의 시선 집중도 상승.

---

## 4. 탭바 — 활성 탭·구분감 강화

**현재:** 활성 탭에 색상만 강조.

**제안:**
- 활성 탭 아이콘/텍스트에 `text-shadow` 또는 ` glow` 추가
- 탭바 배경에 상단 얇은 라인 (1px gradient)으로 구분감 부여
- 선택된 탭 위쪽에 작은 인디케이터 바(2px) 추가

→ 하단 네비게이션의 계층감과 가독성 개선.

---

## 5. 점수 게이지 주변 — 외곽 글로우·입체감

**현재:** 게이지만 눈에 띄고, 주변은 텅 빈 느낌.

**제안:**
- 점수 원형 바깥에 2~3겹 `box-shadow` (같은 색, 다른 blur/opacity)
- 게이지 뒤에 ` radial-gradient` 원형 후광
- 잭팟 시 파티클·글로우 강도와 범위를 더 키워서 “폭발” 느낌 강화

→ 메인 포인트가 더 뚜렷해지고 임팩트가 커짐.

---

## 요약

| 영역         | 추가 내용                                      |
|--------------|-----------------------------------------------|
| 1. 배경      | noise/grain, 별·라인 패턴                     |
| 2. 카드      | 테두리 글로우, 그라데이션 테두리, 호버 효과   |
| 3. 설명 박스 | accent 라인, 블러 강화, 좌측 세로 바          |
| 4. 탭바      | 활성 glow, 상단 라인, 인디케이터 바           |
| 5. 점수 게이지 | 다중 shadow, radial 후광, 잭팟 글로우 강화   |

위 5군데를 적용하면 심플함을 유지하면서 시각적 깊이와 포인트가 더 살아날 것입니다.
