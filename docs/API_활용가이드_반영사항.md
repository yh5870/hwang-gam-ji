# 기상청 API 활용가이드 반영 사항

## ASOS 시간자료 (기상청01)

### 1. 데이터 제공 범위
- **전일(D-1) 자료까지 제공**
- **전일 자료는 조회시간 기준 11시 이후에 조회가능**
- → 11시 이전: 2일 전 23시 데이터 요청
- → 11시 이후: 전일 (현재시각-1)시간 데이터 요청

### 2. 시정(vs) 단위
- **10m 단위** (활용가이드 명시)
- km 변환: `vs / 100` (기존 vs/1000 오류 수정)
- 예: vs=5000 → 50km, vs=310 → 3.1km

### 3. resultCode
- `00` 또는 `0`: 정상 (NORMAL_SERVICE)
- `03`: 데이터없음 (NODATA_ERROR)

## 단기예보 (기상청41)

### resultCode
- `00` 또는 `0`: 정상

### 발표시각
- 0200, 0500, 0800, 1100, 1400, 1700, 2000, 2300 (KST)
- 매 발표시각 30~40분 후 데이터 제공
